# Loan Risk Score Prediction System
**Overview**

This project is an end-to-end Machine Learning application that predicts the financial risk score of loan applicants. It is designed for bank employees to assess loan applications through a secure and user-friendly Streamlit web application.

**Machine Learning**
* Data cleaning and preprocessing
* Feature engineering
* Exploratory Data Analysis (EDA)
* Robust Scaling and Ordinal Encoding
* Random Forest Regression model
* Hyperparameter tuning using RandomizedSearchCV
* Model evaluation using R² Score
* Model serialization using Joblib

**Application Features**
* Employee login authentication
* Create new loan applications
* Save applications as pending
* Resume pending applications
* Predict applicant risk score
* Automatic Loan ID generation
* Dashboard displaying pending and completed applications
* Automatic transfer of processed applications to completed records
* Banking-inspired user interface

**Risk Score Categories**
- Below 35 - Low Risk
- 35 – 65	Moderate Risk
- Above 65-	High Risk
